* Play Information *

- Unzip the contents of dm_deadtrack.zip to your
  valve/steam/steamapps/[username]/half-life 2 deathmatch/maps folder.
- Run Half-Life 2 Deathmatch.
- You can create your own server selecting the map from the list, or join
  an existing one.


* Description *

Game type     	: Deathmatch
Players load   	: 6 - 18, maybe more
Respawn spots   : 38
Known bugs      : None
New Textures    : Some new decal signs, embedded in the .bsp file.


* Notes *

This map was inspired to a real train station here in Rome.
It is rather big, with a wide, open central area and small corridors all around,
and this variety allows for strategic and fluent gameplay.
